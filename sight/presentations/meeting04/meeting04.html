Things We Plan to Explore: AIOps (Jay Shan)

  • Latency Concerns
    • Current architecture may introduce delays in chatbot responses.
    • Transmission speed is critical since chatbot outputs will inform downstream applications (e.g., VR/AR modules).
  • Architecture Optimization
    • Shift from a linear to a hub-and-spoke streaming design to speed up message transmission and processing.
    • Central hub for message coordination, with optimized spokes for LLM queries, RAG lookups, and monitoring.
  • Model Deployment Strategies
    • Evaluate fine-tuning the LLM to reduce reliance on extensive RAG calls.
    • Consider edge deployment for faster inference and lower network latency.
